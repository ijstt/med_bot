# MedBot

Telegram-бот для записи на приём к врачу, поиска медицинских учреждений и получения информации о них.

## Возможности

- Регистрация и авторизация пользователей
- Запись на приём к врачу через интерактивное меню
- Поиск медицинских учреждений по областям и городам
- Отображение учреждений на карте (Яндекс Геокодер)
- Определение возможного диагноза по симптомам (опросники по направлениям: глазной, ЖКТ, стоматология, хирургия, ЛОР)
- Административная панель: управление врачами, учреждениями, областями и городами
- Тестирование знаний пользователей

## Стек технологий

| Компонент | Технология |
|-----------|-----------|
| Язык | Python 3.10+ |
| Telegram API | aiogram 2.25 |
| База данных | SQLite 3 |
| Геокодирование | Яндекс HTTP Геокодер API |
| HTTP-запросы | Requests |

## Установка

```bash
git clone https://github.com/ijstt/med_bot.git
cd med_bot
pip install -r requirements.txt
```

## Настройка

В файле `config.py` укажите:

- `TOKEN` — токен Telegram-бота ([@BotFather](https://t.me/BotFather))
- API-ключ для Яндекс Геокодера

## Запуск

```bash
python main.py
```

## Структура проекта

```
med_bot/
├── main.py          # Основная логика бота: хендлеры, FSM-состояния
├── config.py        # Конфигурация: токен, API-ключи
├── datab.py         # Работа с базой данных (CRUD-операции)
├── keyboard.py      # Клавиатуры и кнопки
├── geocode.py       # Геокодирование адресов через Яндекс API
├── tester.py        # Модуль тестирования
├── hospital.db      # БД с учреждениями, врачами и расписанием
└── requirements.txt # Зависимости
```
